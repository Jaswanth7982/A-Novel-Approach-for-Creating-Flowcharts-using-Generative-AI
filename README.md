# A Novel Approach for Creating Flowcharts using Generative AI

## Project Description

This project is a web application built using **Django** that generates flowcharts automatically using generative AI techniques.

## Features

* Generate flowcharts automatically
* User login and registration
* Admin management system
* Web interface using Django templates

## Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite

## Installation

1. Clone the repository

```
git clone https://github.com/Jaswanth7982/A-Novel-Approach-for-Creating-Flowcharts-using-Generative-AI.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the project

```
python manage.py runserver
```

Then open
http://127.0.0.1:8000
